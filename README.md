## 🚀 Funcionalidades Principais

*   **Check-in Rápido:** Interface simplificada para clientes via `cliente.html`.
*   **Gestão em Tempo Real:** Painel administrativo para controle de serviços via `barbeiro.html`.
*   **Segurança:** Sistema de acesso com validação de senha.
*   **Mobile First:** Design totalmente otimizado para celulares e tablets.

## 🛠️ Tecnologias

*   **Frontend:** HTML5, CSS3 e JavaScript (ES6+).
*   **Hospedagem:** GitHub Pages.
*   **Integração:** Preparado para acesso via QR Code físico no estabelecimento.
